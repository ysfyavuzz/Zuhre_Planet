# Sound Files

Bu klasör bildirim ses dosyalarını içerir.

## Gerekli Ses Dosyaları

1. **message.mp3** - Yeni mesaj bildirimi
2. **notification.mp3** - Genel bildirim sesi
3. **call.mp3** - Arama bildirimi
4. **sent.mp3** - Mesaj gönderildi sesi
5. **error.mp3** - Hata bildirimi
6. **success.mp3** - Başarılı işlem bildirimi

## Ses Özellikleri

- Format: MP3
- Süre: 0.5-2 saniye arası (kısa ve öz)
- Bit rate: 128 kbps
- Frekans: 44.1 kHz
- Stereo
